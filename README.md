# ToDoList앱 만들기 프로젝트

### RxSwift를 활용하여 ToDoList앱 만들기
- [x] RxSwift 비동기 처리
- [x] MVVM 구조로 프로젝트 구현

### Step.1
- 프로젝트 구조 설정
- 라이브러리 의존성 추가

### Step.2
- MemoryStorage 구현
- 기본적인 CRUD 메서드 구현

### Step.3
- Scene 생성(스토리보드)
- ViewController 생성
- ViewModel 생성

### Step.4
- TransitionStyle, TransitionError 생성(화면전환)
- Scene 전환에 필요한 메서드 구현
- Scene 전환을 담당하는 SceneCoordinator 생성 및 프로퍼티 & 메서드 구현

### Step.5
- AppDelegate에서 실행화면 구현
- 불필요한 주석 및 메서드 삭제
- ToDoListViewModel & ToDoListViewController bind 메서드 구현
- CustomCell 생성
- View 디자인은 추후에 적용
